Ui Design PictGram Mobile App

Ui Design PictGram Mobile App

Merupakan Rancangan Ui Design Untuk Mobile App, PictGram adalah Sebuah Apliasi Untuk Mempublishkan Gambar Ke Dunia Maya dan Bisa Juga Untuk Berbagi Gambar Satu Sama Lainnya. Aplikasi Ini Didesign Semaksimal dan Minimalis Mungkin Agar User Yang Menggunaan Mendapatkan Kenyamana Saat Menggunakannya.

Ditampilan awal aplikasi pictgram ini user bisa langsung melihat gambar-gambar yang pernah dimasukan oleh user lain dan dapat untuk melihat secara detail maupun melihat siapa pemilik akun dari gambar tersebut.

Mockup Dari Ui Design PictGram Mobile App
Halaman Utama

Dihalaman Utama User akan langsung di tampilakan dengan foto tau gambar yang sudah di pernah dimasukan oleh user lainnya, di sini user bisa melihat foto atau gambar tersebut sesuai dengan riteria yang sudah di tentukan bawaan dari aplikasi, selain foto atau gambar yang sudah ada, user juga bisa mencari foto atau gambar yang lebih spesifik dengan mengklik tombol pencarian yang ada di float status bar di bagian bawah.

Halaman Detail Foto

Dihalaman Utama User akan langsung di tampilakan dengan foto tau gambar yang sudah di pernah dimasukan oleh user lainnya, di sini user bisa melihat foto atau gambar tersebut sesuai dengan riteria yang sudah di tentukan bawaan dari aplikasi, selain foto atau gambar yang sudah ada, user juga bisa mencari foto atau gambar yang lebih spesifik dengan mengklik tombol pencarian yang ada di float status bar di bagian bawah.

Halaman Profil User

Dihalaman user lain dapat melihat postingan foto atau gambar yang sudah pernah di masukan oleh user tersebut, dan juga user lain dapat mengikuti agar mendapatkan info foto terbaru dari user yang diikuti tersebut.

Halaman pencarian

Selain user dapat menekan pilihan foto yang sudah ada di halaman utama, user juga bisa mencari foto atau gambar drngan kkeyword yang diinginkan oleh user, dan di halaman pencarian juga user juga dapat melihat foto dari jenis yang sudah ada seperti pencarian populer dan pencarian buat anda.

Halaman Gambar Tersimpan

Dihalaman gambar tersimpan adalah foto atau gambar yang sudah di pin atau di simpan oleh user, di gambar tersimpan iniuser juga bisa mengelompokan gambar tersebut dan di jadikan sebuah album.
