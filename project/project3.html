Ui Design PictBook Mobile App

Ui Design PictGram Mobile App

Merupakan Rancangan Ui Design Untuk Mobile App, PictBook adalah Sebuah Apliasi Untuk Membaca Buku Secara Online Atau Offline.

Ditampilan awal aplikasi pictBook ini user bisa langsung melihat halaman utama yang mana dihalaman ini sudah di tampilakan beberapa buku yang bisa di baca oleh user, selain itu di halaman utama ini user bisa melihat buku yang terpopuler dan yang terbaru.

Mockup Dari Ui Design PictGram Mobile App
Halaman Utama

Design di halaman utama ini di harapkan user yang menggunakan apliasi pictbook ini dapat langsung tertuju untuk membaca buku, di karenakan di halaman utama sudah di tampilkan buku-buku yang terpopuler dan juga yang terbaru.

Halaman Detail buku

Dihalaman detail buku ini user bisa melihat sinopsis dari buku yang akan di baca tersebut, selain itu user jga bisa melihat rating dan ulasan dari user lain yang sudah membaca buku tersebut, selain itu user yang melihat detail buku ini juga bisa memberian rating dan ulasan.

Halaman Pin Buku

Dihalaman inibertujuan agar user dapat melohat dan membaca buku yang sebelumnya sudah di baca, kata lain di pin buku ini user dapat menyimpan buku buku yang ingin di baca di lain waktu tanpa harus mencari nya lagi

Halaman pencarian

Selain user dapat memilih pilihan buku yang sudah ada di halaman utama, user juga dapat mencari nama buku di halaman pencarian, untuk membantu user dalam mencari buku , di apliasi ini juga menyediakan recomendasi buku bagi user.

Inspirasi dan Referensi

Referensi yang diambil dari UI Design Mobile App PictGram diambil dari Aplikasi Google play Book.
